## 🔬 분석 방법론

### 1️⃣ Pearson vs Spearman 상관분석

**왜 두 가지를 비교했나?**

순위(1~20위)는 **순서형(ordinal) 변수**로, Pearson보다 Spearman이 더 적절합니다.
두 결과를 비교함으로써 분석의 견고성을 검증했습니다.

| 구분 | Pearson | Spearman |
|------|---------|----------|
| 적합 변수 | 연속형 | 순서형 ✅ |
| xPTS 평균 r | -0.867 | -0.841 |
| 결론 | 두 방법 모두 동일한 패턴 → 분석 견고성 확인 |

**p-value를 함께 보고한 이유**

상관계수만으로는 우연인지 진짜인지 알 수 없습니다.
p < 0.05이면 통계적으로 유의미한 관계임을 의미합니다.

---

### 2️⃣ VIF 다중공선성 분석

**왜 했나?**

xG, xPTS, DC가 서로 높은 상관관계를 보이며, xPTS는 xG 기반으로 산출된 지표입니다.
변수 간 겹침이 크면 분석 결과가 왜곡될 수 있어 VIF로 확인했습니다.

| 지표 | VIF (평균) | 상태 |
|------|-----------|------|
| xPTS | ~140 | ⚠️ 심각한 다중공선성 |
| xG | ~58 | ⚠️ 심각한 다중공선성 |
| xGA | ~31 | ⚠️ 다중공선성 |
| DC | ~8 | ⚠️ 주의 |
| PPDA | ~2.5 | ✅ 정상 |

**해결책: PCA**

5개 지표를 PCA로 압축 → PC1이 전체의 **79.2% 설명**
PC1 = xG(+) + xPTS(+) + DC(+) + xGA(-) + PPDA(-) → **"팀 전반적 강도 지수"**

---

### 3️⃣ K-means 군집 분석

**왜 K-means를 선택했나?**

4가지 방법을 비교한 결과 K-means가 가장 높은 Silhouette Score를 기록했습니다.

| 방법 | Silhouette Score | 결과 |
|------|-----------------|------|
| **K-means (k=3)** | **0.326** | ✅ 최고 |
| 계층적 (Ward) | 0.294 | |
| GMM | 0.254 | |
| DBSCAN | 군집 1개 | ❌ 부적합 |

k=3 선택 근거: 통계적 최적(k=2)보다 PL 실제 경쟁 구조(상/중/하위권)를 반영

---

### 4️⃣ Granger Causality 분석

**왜 했나?**

상관관계 ≠ 인과관계입니다.
"xG가 높으면 순위도 높다" vs "좋은 팀이 xG를 만든다" — 어느 방향인지 검증했습니다.

**결과**

모든 지표 p > 0.05 → 다음 시즌 순위 예측 불가

**해석**

전술 지표는 당해 시즌을 잘 설명하지만 다음 시즌을 예측하지는 못합니다.
따라서 **매 시즌 실시간 모니터링**이 필요하며, 이것이 PL DataHub의 존재 이유입니다.

---

### 5️⃣ 수비의 역설 재검증

"수비 지표의 양의 상관이 단순히 수비 시간 누적 효과 아닌가?"

**검증 방법 및 결과**

PPDA(압박 강도)로 per-game 보정 후 재분석한 결과,
4시즌 모두 양의 상관이 유지되었습니다.

| 시즌 | PPDA vs 순위 (Spearman r) | p-value |
|------|--------------------------|---------|
| 21-22 | +0.396 | 0.084 |
| 22-23 | +0.638 | 0.003 ✅ |
| 23-24 | +0.521 | 0.019 ✅ |
| 24-25 | +0.471 | 0.036 ✅ |

**결론**: 단순 누적 효과가 아닌 실제 전술적 차이 확인
